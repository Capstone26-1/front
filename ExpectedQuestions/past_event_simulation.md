# Q. 과거 사건은 어떻게 시뮬레이션하나요?

## 답변

API별로 과거 조회 가능 여부가 다릅니다.

| API | 과거 조회 | 방법 |
|---|---|---|
| Tmap 대중교통 | ✅ 가능 | `searchDttm=YYYYMMDDHHMM` 파라미터 |
| 네이버 뉴스 | ✅ 가능 | 해당 시점 기사가 그대로 보존됨 |
| ITS 도로 돌발상황 | ❌ 불가 | 실시간 전용 |
| 기상청 | ❌ 불가 | 실시간 전용 |
| 서울 지하철 | ❌ 불가 | 실시간 전용 |

## 서소문 고가도로 붕괴 사례 (2026년 5월 27일)

- Tmap: `searchDttm=202605272300`으로 당일 밤 11시 경로 재현 가능
- 네이버 뉴스: 실제 기사 3건 검색 가능 ("서소문 고가차도 붕괴 당일 '뚝' 소리...")
- ITS: 사고 장기화로 현재도 통제 데이터 잔류 가능

## 한계 및 대응

실시간 전용 API의 과거 데이터는 조회 불가합니다.
발표용 시연이 필요한 경우 해당 시나리오의 데이터를 Mock으로 보완합니다.
